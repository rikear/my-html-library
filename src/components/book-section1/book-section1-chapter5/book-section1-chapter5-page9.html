grid-auto-rows and grid-auto-columns

Let's use grid-auto-rows and grid-auto-columns to make any extra grid items the same size as the existing ones:

my-first-grid {

display: inline-grid;

grid-gap: 10px;

grid-template-rows: 200px 200px;

grid-template-columns: 200px 200px;

grid-auto-rows: 200px;

grid-auto-columns: 200px;

background-color: #e4e4e4;

}

Now, without writing any extra CSS, when additional items are placed in our grid, they get the 200 px x 200 px sizing we have defined. Here, we have added another item in the DOM for a total of six:
