Column reverse

Want them stacked in the opposite direction? Just change to flex-direction: column-reverse; and you 're done.

There is a flex-flow property that is shorthand for setting flex-direction and flex-wrap in one go. For example, flex-flow: row wrap; would set the direction to a row and set the wrapping on. However, at least initially, I find it easier to specify the two settings separately.
