Learn about regular expressions (regexes)

If you've never encountered regular expressions before, I'd suggest starting here: https://en.wikipedia.org/wiki/Regular_expressions.Regular expressions are used across many programming languages as a means of matching strings. While the format is intimidating at first, they are incredibly powerful and flexible. For ex-ample, you could build a regular expression to match a password format or select a certain style of CSS class-naming convention. To help build up your own regex pattern and get a visual understanding of how they work, I'd recommend starting with a browser-based tool like https: //www.regexr.com/.

The following code is an example:

<div>

<label for="name"> Your Name (first and last) </label>

<input

id="name"

name= "name"

pattern="^([\D]{2,30} \s+)+([a-zA-Z]{2, 30})$"

placeholder="Dwight Schultz"

required

/>

</ div>
