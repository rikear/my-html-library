The first value is the skew applied to the x axis (in our example, 40deg), while the second (12deg) is for the y axis. Omitting the second value means any value will merely be applied to the x axis (horizontal). For example:

transform: skew(10deg) ;

Just like with translate, you can apply skews to just one axis with skewx () and skewv().

The matrix transform syntax looks rather impenetrable. Here's our example code:

.matrix:hover {

transform: matrix (1.178, -0.256, 1.122, 1.333, -41.533, -1.989);
