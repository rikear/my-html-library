Wow! Flexbox really makes these kinds of changes trivial. In the example, the value of align-self was set to flex-end. Let's consider the possible values we could use on the cross axis before looking at alignment in the main axis.

Possible alignment values

For cross-axis alignment, Flexbox has the following possible values:

  • flex-start: Setting an element to flex-start makes it begin at the "starting" edge of its flex container.
  • flex-end: Setting to flex-end aligns the element at the end of the flex container.
  • center: This puts it in the middle of the flex container.
  • baseline: This sets all the flex items in the container so that their baselines align.
  • stretch: This makes the items stretch to the size of their flex container (in the cross axis).
