In this example (example_08-03), a direction has not been specified, so the default top-to-bottom direction applies.

Color stops inside a gradient are written comma-separated and defined by giving first the color, and then the position of the stop. It's generally advisable not to mix units in one notation, but you can. You can have as many color stops as you like and colors can be written as a keyword, hex, RGB, HSL, or any other color format we looked at in the previous chapter.

You can read the W3C specification for linear background gradients at https:// www. w3. org/TR/css-images-3/#linear-gradients.
