The tel input type

You can set an input field to expect a telephone number type of value, like this:

type="tel"

Here's a more complete example:

<div>

<label For="tel"> Telephone (so we can berate you if you're wrong) </label >

<input

id="tel"

name="tel"

type="tel"

placeholder="1-234-546758"

autocomplete="off"

required

/>

</div>

Browsers do little validation on the tel innut tupe. When an incorrect value is indut, they fail to provide a suitable warning message.
