Let's suppose I only wanted to select "indoor swimming." I can't use just data-location="indoor" as that would get the first element too. I can't use data-activity-name="swimming" as that would get me the second and the fourth, but I can do this:

[data-activity-name="swimming"] [data-location="indoor" ] {

/*styles */

}

This selects elements that have "swimming" as the activity name, as long as they also have " indoor" as the location.
