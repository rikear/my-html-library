Let's use the alignment properties we learned about in the last chapter to try centering our grid items:

.my-first-grid {

display: grid;

grid-gap: 10px;

grid-template-rows: 200px 200px;

gria-template-columns: 200px 200px;

background-color: #e4e4e4;

align-items: center;

justify-content: center;

}

When I first started with Grid Layout and tried something like this, I expected to see the numbers perfectly centered in their respective grid tracks. However, that is not the case:

Figure 5.3: If you don't set a width, a grid will consume the available space

If we think about what we have done, it does make sense. We made a grid, with two columns and two rows, each 200 px, and asked for the items to be both vertically and horizontally centered. Because we have used grid and not inline-grid, the grid fills the entire width of the page, despite the fact that our grid items don't need all that space.

my-first-grid {

display: inline-grid;

grid-gap: 10px;

grid-template-rows: 200px 200px;

}
